รายละเอียดหลักสูตร 11 Blocks

ปลดล็อกเวลา 2–3 ชั่วโมง/วัน ด้วย Claude — จาก Zero สู่ Workflow ที่ทำงานแทนคุณได้จริง

หลักสูตรนี้ต่างจากคอร์ส AI ทั่วไปอย่างไร

คอร์สส่วนใหญ่สอนให้ “ถาม-ตอบ” กับ AI แต่ 88% ของคนใช้ AI แค่ค้นข้อมูล มีเพียง 5% ที่ใช้เปลี่ยนวิธีทำงานจริง (EY 2025) หลักสูตรนี้พาคุณข้ามเส้นจาก “ผู้ใช้” ไปเป็น “ผู้สั่งงาน” — เรียนแบบลงมือทำกับงานจริงของตัวเอง ทุก Block มีผลลัพธ์ที่จับต้องได้กลับไปใช้ทันที และจบวันด้วย Workflow อัตโนมัติตัวแรกที่ทำงานแทนคุณได้จริง

Part 1Foundation — เริ่มจาก 0 สู่ใช้งานได้จริงBlock 01–07
01Claude คืออะไร? — AI Colleague ไม่ใช่แค่ Chatbot30 นาที
จะได้เรียนรู้อะไร
  • เข้าใจความต่างระหว่าง “Chatbot ที่ตอบคำถาม” กับ “AI Colleague ที่ทำงานให้” และทำไม mindset นี้เปลี่ยนวิธีทำงานทั้งหมด
  • รู้ว่า Claude ทำอะไรได้จริงในบริบทงานออฟฟิศ: เขียน · วิเคราะห์ · วางแผน · ผลิตเอกสาร
  • มองเห็นว่างานในแต่ละวันของตัวเองส่วนไหนที่มอบให้ Claude ทำแทนได้ทันที
กิจกรรมในคลาส
  • ดูตัวอย่างงานจริงที่ Claude ทำได้ เทียบกับวิธีเดิมที่ใช้เวลามากกว่า
ได้กลับไปใช้ทันที ›เปลี่ยนมุมมองจาก “ถาม AI” เป็น “สั่ง AI ให้ทำงานแทน” — รากฐานของทั้งหลักสูตร
02First Time Setup — ตั้งค่าให้พร้อมใช้จริง40 นาที
จะได้เรียนรู้อะไร
  • ติดตั้ง Claude Desktop บนเครื่องตัวเอง และรู้จักส่วนประกอบสำคัญของหน้าจอ
  • เปิด Capabilities และเข้าใจว่าแต่ละอันเปิดใช้ตอนไหน
  • ตั้ง Global Instructions ให้ Claude รู้จักคุณ บทบาท และสไตล์งานที่ต้องการ
  • รู้จัก Skills — ชุดความสามารถเฉพาะทางที่เรียกใช้ซ้ำได้
กิจกรรมในคลาส
  • ตั้งค่าเครื่องตัวเองจนใช้งานได้จริง จบภายในคลาส — ไม่ต้องกลับไปทำเองที่บ้าน
ได้กลับไปใช้ทันที ›เครื่องพร้อมใช้งานเต็มรูปแบบ: Desktop + Instructions + Skills ตั้งค่าเสร็จในคลาส
03Prompt 101 — สูตร CTFC เขียน Prompt ให้ได้งานทุกครั้ง45 นาที
จะได้เรียนรู้อะไร
  • เข้าใจว่าทำไม prompt เดิม ๆ ถึงได้ผลลัพธ์กว้าง ๆ ไม่ตรงใจ
  • เรียนสูตร CTFC — โครงเขียน prompt 4 องค์ประกอบที่ทำให้ได้ผลลัพธ์แม่นยำและใช้ซ้ำได้
  • ฝึกแปลงคำสั่งลอย ๆ ให้เป็น prompt ที่ให้งานคุณภาพตั้งแต่ครั้งแรก
กิจกรรมในคลาส
  • เขียน prompt จริงจากงานของตัวเอง แล้วเทียบผลลัพธ์ก่อน–หลังใช้สูตร CTFC
ได้กลับไปใช้ทันที ›สูตร CTFC ที่หยิบไปเขียน prompt ได้ทุกงาน พร้อม Cheatsheet กลับบ้าน
04Use Case Workshop — 4 ประเภทงานหลัก45 นาที
จะได้เรียนรู้อะไร
  • รู้จัก 4 กลุ่มงานที่ Claude ช่วยได้ดีที่สุด: Writing · Analysis · Planning · Document Production
  • เห็นแนวทางและ prompt ตัวอย่างของแต่ละประเภทงาน
  • รู้ว่างานของตัวเองอยู่กลุ่มไหน และเริ่มใช้ Claude กับงานนั้นอย่างไร
กิจกรรมในคลาส
  • ลงมือทำงานจริงอย่างน้อย 1 ชิ้นจากงานของตัวเอง ในกลุ่มที่เลือก
ได้กลับไปใช้ทันที ›ได้ผลงานจริง 1 ชิ้น + แนวทางใช้ Claude ครอบคลุมงานหลักในแต่ละวัน
05Claude in Excel & PowerPoint — สั่งงานข้าม App35 นาที
จะได้เรียนรู้อะไร
  • ใช้ Add-in ของ Claude ที่ฝังอยู่ใน Excel และ PowerPoint โดยตรง
  • สั่งงานในเครื่องมือที่ใช้อยู่ทุกวัน โดยไม่ต้องสลับหน้าจอไปมา
  • สร้างและปรับตาราง สูตร และสไลด์ ให้ได้ผลลัพธ์ที่ต้องการจากคำสั่งภาษาคน
กิจกรรมในคลาส
  • ทดลองสั่งงานใน Excel/PowerPoint จากไฟล์ตัวอย่าง จนได้ผลลัพธ์ตามต้องการ
ได้กลับไปใช้ทันที ›ทำงานใน tool ที่คุ้นเคยได้เร็วขึ้น สั่งงานข้าม app ได้โดยไม่เสีย context
06Iterate & Trust — 3 Moves ปรับผลลัพธ์ให้ตรงใจ45 นาที
จะได้เรียนรู้อะไร
  • เข้าใจว่าผลลัพธ์ครั้งแรกไม่จำเป็นต้องสมบูรณ์ — การ iterate คือทักษะหลัก
  • เรียน 3 Moves สำหรับดันผลลัพธ์จาก “พอใช้” ให้เป็น “ใช้ได้จริง”
  • สร้างความมั่นใจในการตรวจสอบและไว้ใจงานของ Claude อย่างมีวิจารณญาณ
กิจกรรมในคลาส
  • ฝึกปรับผลลัพธ์ของตัวเองด้วย 3 Moves จนได้คุณภาพที่ส่งงานได้
ได้กลับไปใช้ทันที ›เทคนิค 3 Moves ที่ทำให้ได้งานตรงใจ โดยไม่ต้องเริ่มเขียนใหม่ทุกครั้ง
07Recap + Q&A — ทบทวนและปูทางสู่ Cowork15 นาที
จะได้เรียนรู้อะไร
  • สรุปสิ่งที่ได้จาก Part 1 ให้เชื่อมเป็นภาพเดียว
  • เคลียร์ข้อสงสัยก่อนก้าวสู่การสร้างระบบอัตโนมัติใน Part 2
  • เห็นภาพว่าทักษะพื้นฐานที่เพิ่งเรียน จะต่อยอดเป็น Workflow ได้อย่างไร
กิจกรรมในคลาส
  • ถาม-ตอบสด และตรวจ checklist ความพร้อมก่อนเข้า Part 2
ได้กลับไปใช้ทันที ›พื้นฐานแน่น พร้อมต่อยอดสู่การสร้างระบบที่ทำงานแทนคุณ
Part 2Cowork — สร้างระบบที่ทำงานแทนคุณBlock 08–11
08Cowork Deep Dive — เลือก Chat หรือ Cowork ให้ถูก65 นาที
จะได้เรียนรู้อะไร
  • เข้าใจความต่างระหว่าง Chat mode (สนทนา) กับ Cowork mode (ทำงานกับไฟล์และระบบจริง)
  • รู้ 9 สิ่งที่ Cowork ทำได้ ซึ่ง Chat ธรรมดาทำไม่ได้ เช่น จัดการไฟล์ รันงานหลายขั้นตอน เชื่อมต่อเครื่องมือ
  • ตัดสินใจได้ว่างานแบบไหนควรใช้ Chat และงานแบบไหนต้องใช้ Cowork
กิจกรรมในคลาส
  • ทดลองใช้ Cowork กับงานจัดการไฟล์/เอกสารจริง เห็นพลังของ mode นี้กับตา
ได้กลับไปใช้ทันที ›เลือก mode ได้ถูกทุกครั้ง — ไม่เสียเวลากับเครื่องมือที่ไม่เหมาะกับงาน
09RAPID Framework — เลือกว่างานไหนควร Automate30 นาที
จะได้เรียนรู้อะไร
  • รู้จัก RAPID Framework — เกณฑ์คัดกรองว่างานไหนคุ้มค่าที่จะทำเป็นระบบอัตโนมัติ
  • ประเมินงานของตัวเองด้วยเกณฑ์ที่จับต้องได้ ไม่ใช่ความรู้สึก
  • จัดลำดับความสำคัญ เลือกงานที่ให้ผลตอบแทนสูงสุดมาทำก่อน
กิจกรรมในคลาส
  • ให้คะแนนงานของตัวเองด้วย RAPID แล้วเลือก 1 งานไปสร้างเป็น Workflow ใน Block ถัดไป
ได้กลับไปใช้ทันที ›เกณฑ์ RAPID ที่ใช้เลือกงาน automate ได้เอง ไม่หลงลงทุนผิดจุด
10Build Your First Workflow — สร้าง Workflow ตัวแรก90 นาที
จะได้เรียนรู้อะไร
  • สร้าง Project ให้ทำหน้าที่เป็น sub-agent (ผู้ช่วยเฉพาะทาง) ที่ทำงานซ้ำ ๆ แทนคุณ
  • วางขั้นตอน Workflow ทีละ step จนระบบทำงานได้ครบวงจร
  • ทดสอบ ปรับแก้ และทำให้ Workflow เชื่อถือได้พอที่จะใช้กับงานจริง
กิจกรรมในคลาส
  • ลงมือสร้าง Workflow ตัวแรกจากงานจริงของตัวเอง (Block ที่ยาวและลงมือมากที่สุดของทั้งวัน)
  • ทดสอบรันจริงและปรับจูนกับผู้สอนแบบตัวต่อตัว
ได้กลับไปใช้ทันที ›Workflow อัตโนมัติ 1 ตัว พร้อมใช้จริง กลับไปทำงานแทนคุณได้ตั้งแต่วันรุ่งขึ้น
11Showcase + Daily Habit — ทำให้ใช้ได้ต่อเนื่อง25 นาที
จะได้เรียนรู้อะไร
  • นำเสนอ Workflow ของตัวเอง และเรียนรู้จากของเพื่อนร่วมคลาส
  • รู้จัก 10-Minute Daily Habit — วิธีทำให้การใช้ AI กลายเป็นนิสัยประจำวัน
  • ใช้ ROI Tracker วัดเวลาที่ประหยัดได้จริง และรับ 30-Day Challenge ไปทำต่อ
กิจกรรมในคลาส
  • Demo Workflow ของตัวเอง + วางแผน Daily Habit และเป้าหมาย 30 วันข้างหน้า
ได้กลับไปใช้ทันที ›แผนใช้งานต่อเนื่อง + ROI Tracker + 30-Day Challenge ที่ทำให้ทักษะไม่หายไปหลังคลาส
